# ImageBinarizer

Курсовой проект посвящён изучению и практическому применению потоков данных в C# на примере разработки Windows Forms приложения для обработки изображений.

## Цель проекта
Показать эффективность использования потоков данных, асинхронного программирования и оптимизации памяти при создании производительных и отзывчивых приложений на C#.

## Теоретическая часть
В отчёте рассматриваются:
- основные типы потоков в C#: `FileStream`, `MemoryStream`, `NetworkStream`, `BufferedStream`, `CryptoStream`;
- принципы асинхронных потоков с использованием `async/await`;
- буферизация как способ повышения производительности при работе с большими объёмами данных.

## Практическая часть
Разработано Windows Forms приложение, которое позволяет:
- загружать изображение;
- выбирать цветовой канал (красный, зелёный или синий);
- задавать пороговое значение;
- выполнять пороговую обработку изображения с преобразованием в чёрно‑белый формат.

Для предотвращения зависания интерфейса обработка изображений реализована в асинхронном потоке с использованием `Task.Run()`.  
Также применяются оптимизированные методы работы с пикселями (`BitmapData`) для ускорения обработки изображений, включая файлы большого размера.

## Требования
- Windows;
- .NET Framework 4.7.2;
- Visual Studio 2022 (или совместимая).

## Запуск
1. Откройте решение `ImageBinarizer.sln` в Visual Studio.
2. Соберите и запустите проект.

## Использование
1. Нажмите «Загрузить изображение…».
2. Введите порог.
3. Выберите канал R/G/B.
4. Нажмите «Преобразовать!».

## Структура проекта
- `ImageBinarizer\Form1.cs` — логика загрузки и преобразования.
- `ImageBinarizer\Form1.Designer.cs` — разметка формы.
- `ImageBinarizer\Program.cs` — точка входа.
